# Spec Driven Development

---

## 🚀 Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/Rinkesh-Ranpariya-Simform/Spec-Driven-Development.git
cd Spec-Driven-Development

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```
